### 비선형 문제
- 선형 자료구조로 표현 불가능한 문제들
- 예시: 계층적 문제, 순환 종속성 문제
- 데이터가 선형(일직선)으로 저장되지 않고, 하나의 데이터에서 여러 데이터들 중 하나로 접근할 수 있는 경우

### 계층적 문제
- 어떤 구성이 계층적인 구성을 이루는 경우
- 예시: 회사 조직도, 대학 교과과정 계층도

### 순환 종속성
- 예시: 친구 관계
- 하나의 노드에서 여러 다른 노드로 접근할 수 있는 경우
- 여러 갈래 형태로 갈라질 수 있으므로, 선형 형태가 아님. 따라서 선형 자료구조로 나타낼 수 없음

### 트리: 상하 반전된 형태
- 노드와 노드 사이를 연결하는 간선(edge)를 사용해서 트리를 구성할 수 있음(나무 형태)
- 루트 노드: 트리의 중심이 되는 노드이며, 트리 구조의 맨 위에 있는 노드를 말함
- 실제 나무와 달리 뿌리(루트 노드)가 맨 위에 있으므로 상하 반전된 형태임

### 트리 순회 방식
1. 전위 순회
-> 1. 현재 노드를 방문
-> 2. 현재 노드의 왼쪽 하위 노드를 방문
-> 3. 현재 노드의 오른쪽 하위 노드를 방문
-> 4. 위 과정을 재귀적으로 반복
2. 중위 순회
-> 1. 현재 노드의 왼쪽 하위 노드를 방문
-> 2. 현재 노드를 방문
-> 3. 현재 노드의 오른쪽 하위 노드를 방문
-> 4. 위 과정을 재귀적으로 반복
3. 후위 순회
-> 1. 현재 노드의 왼쪽 하위 노드를 방문
-> 2. 현재 노드의 오른쪽 하위 노드를 방문
-> 3. 현재 노드를 방문
-> 4. 위 과정을 재귀적으로 반복
